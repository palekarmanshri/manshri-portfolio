# 🌐 Personal Portfolio Website

A modern and responsive personal portfolio website showcasing my projects, skills, and experience as a Computer Science student and Frontend Developer. This website reflects my journey, creativity, and passion for building user-friendly web applications.

---

## 🚀 Live Demo

🔗 https://palekarmanshri.github.io/manshri-portfolio/

---

## ✨ Features

* 🌟 Responsive design (Mobile, Tablet, Desktop)
* 🎨 Clean and modern UI/UX
* 📂 Projects showcase section
* 🧑‍💻 About Me section
* 📞 Contact form / social links
* ⚡ Smooth scrolling and navigation
* 🔥 Lightweight and fast performance

## ⚙️ Installation & Setup

1. Clone the repository

```
git clone https://github.com/palekarmanshri/manshri-portfolio.git
```

2. Navigate to the project folder

```
cd manshri-portfolio
```

3. Open `index.html` in your browser
